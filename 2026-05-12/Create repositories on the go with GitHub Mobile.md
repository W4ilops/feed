### Main Topic: GitHub Mobile Creation

**Main Thesis:** You can now easily create new projects (repositories) directly using the GitHub mobile app on both iOS and Android.

**Simple Summary:** The GitHub mobile app allows users to create new project spaces directly from their phone. When creating a project, you can choose if it is public or private, give it a name, and add a description. You also have options to set up basic files like a README to start your project immediately.

**The Bottom Line:** > Creating projects is now much easier because you don't need a computer; you can start building and organizing code right from your mobile device.
