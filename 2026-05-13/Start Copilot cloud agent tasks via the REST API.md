### Main Topic: Copilot Automation

**Main Thesis:** A new API allows users to programmatically use Copilot agents to automate complex tasks across multiple projects.

**Simple Summary:** Copilot agents can work in the background to make and validate code changes. A new API lets users connect these agents to custom scripts, making it easy to automate big jobs like updating many code files, setting up new projects, or preparing weekly software releases automatically.

**The Bottom Line:** > This new tool lets teams use Copilot to handle repetitive, large-scale work automatically, saving significant time on development and setup tasks.
