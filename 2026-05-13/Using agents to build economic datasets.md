### Main Topic: Data Construction

**Main Thesis:** Using AI can make the expensive process of building economic datasets much faster and cheaper.

**Simple Summary:** Researchers found that creating large datasets from public information is very costly. They introduced a method called DRIL, which uses AI agents to automatically gather and organize data from public sources. This process can quickly assemble complex economic datasets, saving significant time and money.

**The Bottom Line:** > This new method shows that using AI to build data can fundamentally change how economic research is done and made.
