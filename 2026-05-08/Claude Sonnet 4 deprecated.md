### Main Topic: Model Update
**Main Thesis:** GitHub is removing an older AI model and telling users to switch to a newer one.
**Simple Summary:** GitHub is stopping use of an older AI helper called Claude Sonnet 4. They want everyone to use a newer version, Claude Sonnet 4.6, instead. If you use GitHub’s AI tools, you need to change how you use them.  If you manage the tools for a company, you’ll need to make sure the new AI model is allowed.
**The Bottom Line:** > Don’t use the old AI model anymore – switch to the updated version to keep your GitHub AI tools working smoothly. If you’re a company administrator, make sure the new AI model is enabled for your team.
